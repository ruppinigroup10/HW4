Game Genre Classifier - Quick Summary

Hey! This is a simple breakdown of how to add an AI-based Game Genre Classifier to a our main page using the facebook / bart-large-mnli from Hugging Face API. Basically, it figures out the genre of a game based on its description.

What It Does

  • Uses an AI model to classify game genres automatically.

How We Set It Up?

  1. API Key: Sign up at Hugging Face and create a token.
  2. Configure It: Add this to appsettings.json: 
    3. {
  "HuggingFace": { "ApiKey": "your-api-key-here" }
}
  3. Backend Work: Create a C# class to send game descriptions to Hugging Face.
  4. Update Database: Add a Genre column to store the results.
  5. Front-End: Add a button to let users classify games.

Example JavaScript Code

function classifyGameGenre(gameId, description) {
    $.ajax({
        url: "YOUR_API_URL/Games/classify-genre",
        type: "POST",
        data: JSON.stringify({ AppID: gameId, Description: description }),
        contentType: "application/json",
        success: function(response) {
            $(`#genre-${gameId}`).text(response.genre);
        },
        error: function(xhr) {
            console.error("Error classifying genre:", xhr.responseText);
        }
    });
}

Issues we faced & Fixes

  • API Key Issues (401 Error): Check if the key is added correctly in appsettings.json.
  • Slow Response: First request can be slow; try adding a timeout.
  • Genre Always Unknown: Make sure descriptions aren’t empty.
